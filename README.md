node_hpit
=========

node.js library for hpit

How to use:

var hpit = require('./node_hpit');

**Plugin**


  1. get the constructor of a plugin

    hpit.plugin
  
  
**Tutor**

  1. get the constructor of a tutor

    hpit.tutor

*Important*

  Checkout the examples folder to learn how to write a tutor or a plugin.
